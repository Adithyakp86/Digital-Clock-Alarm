# Smart Digital Clock with Alarm

A comprehensive digital clock application with alarm functionality built using HTML, CSS, and JavaScript.

## Features

- Real-time digital clock with smooth updates
- Multiple alarm support with individual toggles
- 12/24 hour format toggle
- Local storage for alarm persistence
- Responsive design for all devices
- Dark/Light mode toggle
- Snooze functionality
- Visual and audio notifications
- Complete multi-page website with navigation

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Web Audio API
- LocalStorage API
- Notification API

## Setup Instructions

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Navigate through the different pages using the header navigation

## Usage

- **Home Page**: Displays the main digital clock and project description
- **Clock Page**: Shows the large digital clock with 12/24 hour format toggle
- **Alarm Page**: Set and manage alarms with the form and alarm list
- **About Page**: Information about the project and technologies used
- **Contact Page**: Simple contact form with validation
